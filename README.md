# ReactJS-Project
ReactJS Project

    This repo consist of 15 React Projects from basics to the pro level. This application is created using Vite+React.
        Commands:
            To create new app
                > npm create vite@latest <project-name>
            To install dependencies
                > npm install
            To run app
                > npm run dev
